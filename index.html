UE4 C++ 变量宏

Posted on 2021-03-07

UE4 C++ 变量常见宏

书写格式

Category = “A|B”

创建分类，该分类会出现在该类的 ClassDefaults 下

| 后的为子分类，也可以不适用
| 即定义嵌套层级

BlueprintReadWrite

设置该变量属性为蓝图可读可写，同时给予 Read / Write 权限。
可以在蓝图中调用 Get / Set 方法

BlueprintReadOnly

只提供 Read权限
可调用Get方法

BuleprintWriteOnly

只提供Write权限
可调用Set方法

EditAnyWhere

将变量暴露在信息面板中，以便随时修改变量（无需Set）

EditDefaultsOnly

只能从ClassDefaults面板编辑，无法在Detail面板编辑
只能在编辑器中，打开该蓝图修改参数，但无法在场景中选中该BP进行修改参数

VisibleAnyWhere

将变量暴露在面板中，但不能修改
